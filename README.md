# Minesweeper SQL Project

## Features

- Pygame Minesweeper
- MySQL database integration
- Save scores
- Leaderboard system
- Restart game
- SQL CRUD operations

## Controls

- Left Click = Reveal tile
- Right Click = Flag tile
- R = Restart game
- L = Show leaderboard

## Install

pip install -r requirements.txt

## Run

python minesweeper.py
